项目技术栈

- 前端：Vue.js 2 + Vue Router 3 + Element UI 2
- 后端：Node.js + Express + MySQL
数据库设计 （5张核心表）

- lifts - 升降机基础信息表
- run_records - 运行记录表
- maintenance_records - 维护保养表
- users - 用户权限表
- alarms - 报警记录表
核心功能模块

模块 功能描述 实时监控 升降机状态卡片展示、实时数据更新（每3秒）、详情弹窗、载重比例进度条 统计分析 运行次数/平均时长/总载重统计卡片、报警类型分布柱状图、维护状态饼图、运行记录列表 维护管理 维护记录增删改查、状态流转（待处理→进行中→已完成）、按升降机/状态筛选 用户管理 用户增删改查、角色权限控制（管理员/操作员/维护人员）、用户启用/禁用 报警预警 报警级别分类（警告/严重/紧急）、实时刷新、单个处理/批量处理
